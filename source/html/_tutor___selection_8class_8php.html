◆ printTutorReservationStatus()

printTutorReservationStatus ( )

This function will return a string containing the tutor confirmation information for a specified reservation. Must call getTutorReservationInfo() first

Returns
A string of html will be returned depending on what TutorConfirmation is equal to. 0 results in a simple "Add Tutor" button. 1 results in "(Tutor Name) has confirmed" string. 2 results in "TUTOR CANCELLED - ADD A NEW ONE" button. 3 results in "Waiting for (Tutor's Name) to confirm" string.
